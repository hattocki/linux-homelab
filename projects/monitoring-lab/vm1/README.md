# VM1 — Linux сервер под мониторинг

VM1 используется как контролируемый Linux-сервер, с которого собираются системные метрики для системы мониторинга Prometheus + Grafana.

На сервере установлен Node Exporter — агент мониторинга, который собирает информацию о состоянии операционной системы и предоставляет её Prometheus.

Роль VM1 в архитектуре:

```
VM1 (Ubuntu)

Node Exporter :9100

        ↓

Prometheus (VM2)
```

Установленный компонент:

Node Exporter

Node Exporter используется для сбора следующих метрик:

- загрузка CPU;
- использование оперативной памяти;
- состояние файловой системы;
- сетевой трафик;
- информация о состоянии системы.

Сервис работает через systemd и настроен на автоматический запуск после перезагрузки системы.

Используемый порт:

```
9100
```

Проверка работы Node Exporter:

```bash
curl http://localhost:9100/metrics
```

Управление сервисом:

Проверка статуса:

```bash
systemctl status node_exporter
```

Запуск:

```bash
systemctl start node_exporter
```

Остановка:

```bash
systemctl stop node_exporter
```

Перезапуск:

```bash
systemctl restart node_exporter
```

Проверка автозапуска:

```bash
systemctl is-enabled node_exporter
```

В процессе настройки были проверены:

- доступность метрик Node Exporter;
- подключение VM1 к Prometheus;
- корректность передачи системных метрик;
- работа сервиса после перезагрузки.
