# VM2 — Сервер мониторинга

VM2 используется как сервер мониторинга, на котором работают компоненты сбора и обработки метрик.

Основные задачи VM2:

- получение метрик от контролируемых серверов;
- хранение временных рядов;
- выполнение PromQL запросов;
- подготовка данных для визуализации в Grafana.

В архитектуре проекта VM2 взаимодействует с VM1:

```
VM1

Node Exporter :9100

        ↓

VM2

Prometheus :9090
```

Установленный компонент:

Prometheus

Prometheus выполняет периодический сбор метрик через механизм scrape.

Основные функции:

- сбор данных от Node Exporter;
- хранение метрик;
- выполнение запросов PromQL;
- контроль состояния targets.

Порт:

```
9090
```

Конфигурация Prometheus находится в папке:

```
prometheus/
```
