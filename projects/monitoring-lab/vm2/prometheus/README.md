# Prometheus Configuration

В данной папке находится конфигурационный файл Prometheus.

Файл:

```
prometheus.yml
```

используется для настройки:

- глобальных параметров Prometheus;
- интервала сбора метрик;
- списка monitored targets.

В проекте Prometheus собирает метрики с VM1 через Node Exporter.

Источник метрик:

```
VM1:9100
```

Проверка состояния targets выполняется через Web интерфейс Prometheus:

```
http://VM2_IP:9090/targets
```

Основные элементы конфигурации:

- `global` — общие настройки;
- `scrape_configs` — список источников метрик;
- `job_name` — название группы мониторинга;
- `targets` — адреса серверов для сбора метрик.
