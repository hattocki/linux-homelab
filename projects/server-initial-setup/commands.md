# Commands Used

## System update
```bash
sudo apt update
sudo apt upgrade -y
```

## User management
```bash
sudo adduser adminuser
sudo usermod -aG sudo adminuser
```

## System info
```bash
whoami
hostname
uname -a
```

## Network check
```bash
ip a
ping -c 4 google.com
```

## Disk usage
```bash
df -h
```
